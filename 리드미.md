# 프로젝트3 멀티 에이전트

전세계약 위험 진단/방어 보조 에이전트의 LangGraph 기반 멀티 에이전트 skeleton입니다.

현재 입력 범위는 **전세계약서 1개만 필수 업로드**하는 것으로 축소되었습니다. 등기부등본과 중개대상물 확인설명서는 직접 분석하지 않고, 계약서만으로 확인할 수 없는 필수 확인 항목으로 리포트에 안내합니다.

## 핵심 구조

```text
Analysis Graph
계약서 텍스트, 특약, 주소/보증금 기반 시세 데이터를 분석해 위험 리포트를 생성합니다.

Q&A Graph
분석이 끝난 뒤 사용자의 자유 질문에 ReAct 방식으로 필요한 도구를 선택해 답변합니다.
```

## 현재 Graph

```text
Contract Intake
→ Contract Parser
→ Contract Field Extraction
→ Special Clause Analyzer
→ Market Analyzer
→ Risk Judge
→ Report Writer
```

## MCP 예정 도구

```text
RAG Search MCP
- 법령/가이드/위험 특약/방어 특약/등기부 확인 체크리스트 검색

Market Data MCP
- 전세/매매 실거래가 기반 시세 및 전세가율 분석

Contract Evidence MCP, 선택
- 계약서 원문/특약사항 근거 검색
```

## 실행 예시

```powershell
python -m app.graphs.analysis_graph
python -m app.graphs.qa_graph
```
